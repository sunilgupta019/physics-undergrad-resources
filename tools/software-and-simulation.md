# 🧪 Software & Simulation (Undergraduate Physics)

Free tools to build intuition and computational skills.

## Simulations
| Tool | Use | Link |
|---|---|---|
| PhET Interactive Simulations | Concept simulations (mechanics, E&M, QM) | https://phet.colorado.edu |
| nanoHUB | Nanoscience & device simulations | https://nanohub.org |
| Falstad applets | Circuits, waves, E&M fields | https://www.falstad.com |
| myPhysicsLab | Interactive mechanics | https://www.myphysicslab.com |

## Programming & numerical tools
| Tool | Use | Free course |
|---|---|---|
| Python (NumPy/SciPy/Matplotlib) | Numerical physics, plotting | 🏅 freeCodeCamp Scientific Computing with Python |
| SciPy Lecture Notes | Scientific Python crash course | https://scipy-lectures.org |
| Jupyter | Interactive notebooks | https://jupyter.org |
| SymPy | Symbolic math | https://www.sympy.org |
| GNU Octave | Free MATLAB-like environment | https://octave.org |

## Getting started
Begin with Python + Jupyter, reproduce textbook plots (projectile motion, fields, wavefunctions), then move to small simulations. See [computational course file](../courses/computational.md).
