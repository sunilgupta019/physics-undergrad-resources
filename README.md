# 🎓 Undergraduate Physics — Free Resource Hub

![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg) ![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg) ![Maintained](https://img.shields.io/badge/Maintained-yes-green.svg) ![Resources](https://img.shields.io/badge/resources-100%2B-blue.svg)

A curated, open index of the best **free** courseware, open textbooks, problem sets, and simulations for **B.Sc. / B.Tech. Physics** students. This README is a scannable index — each subject links to a deep list in [`courses/`](courses/).

## 📖 How to use this repo
1. Read the **[study roadmap](roadmap.md)** for a suggested semester-by-semester sequence.
2. Pick a subject below and open its file in [`courses/`](courses/) for the full resource list.
3. Browse [`books/open-textbooks.md`](books/open-textbooks.md) for free textbooks and [`tools/software-and-simulation.md`](tools/software-and-simulation.md) for simulations.

## 🧭 Legend
🎥 video · 📄 notes/PDF · 📝 problem sets · 🧪 labs/sims · 🆓 fully free
