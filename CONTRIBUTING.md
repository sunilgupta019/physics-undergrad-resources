# Contributing

Thanks for helping improve this resource hub! 🙌

## How to suggest a resource
1. Open an **Issue** describing the resource (title, link, provider, level) or open a **Pull Request** editing the relevant `courses/<subject>.md` file.
2. Make sure the resource is **free** (fully open or audit-free) and high quality.
3. Use the existing table format: `Resource | Provider | Format | Level | Notes`.
4. Prefer official course pages over re-uploads.

## Formatting guidelines
- Keep the README a curated index; put deep lists in `courses/`.
- Use the legend icons: 🎥 video · 📄 notes/PDF · 📝 problem sets · 🧪 labs/sims · 🆓 fully free.
- One resource per row; keep descriptions short.

## Quality bar
- Working links only.
- No paywalled-only content unless a free audit track exists.
- Cite the original institution/author.
